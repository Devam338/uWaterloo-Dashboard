# UW Dashboard React

A responsive React dashboard that organizes essential University of Waterloo resources into one searchable, categorized interface.

## Features
- Search resources by name or description
- Filter by Academic, Student Life, and Tools
- Quick stats cards for category counts
- Responsive card-based dashboard UI
- Modular React component structure

## Tech Stack
- React
- Vite
- CSS

## Getting Started
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```

## Project Structure
```bash
src/
  components/
  data/
  App.jsx
  main.jsx
  styles.css
```

## Notes
This project was rebuilt from a static HTML reference into a React application so it is cleaner to maintain, easier to extend, and more aligned with a modern front-end workflow.
